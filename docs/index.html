About SuperSearch v3

What is this?

SuperSearch maps the known superconductor composition space using dimensionality reduction (PCA + UMAP) on 21,263 materials from the UCI Superconductivity Database. Each point represents a named material, positioned by its elemental composition and colored by critical temperature.

Complementing GNoME Predictions

Google DeepMind's GNoME predicted 380,000+ stable crystal structures, but stability differs from superconductivity. SuperSearch identifies thin experimental coverage — gaps that GNoME candidates could fill.

  • GNoME provides candidate structures
  • SuperSearch shows where to look first
  • Together: targeted synthesis priorities

Multi-Factor Opportunity Scoring

Regions are scored on four axes: sparseness (40%), proximity to high-Tc materials (30%), compositional novelty (20%), and synthesis feasibility (10%). Higher scores indicate better synthesis candidates.

Methodology

1. Data Source

SuperSearch uses the UCI Superconductivity Data Set (Hamidieh, 2018), containing 21,263 superconducting materials with experimentally measured critical temperatures (Tc) ranging from 0.00021 K to 185.0 K (mean 34.4 K).

Each material is described by 81 compositional features derived from element fractions multiplied by atomic properties. The dataset does not include crystal structure information; all features are composition-based.

DOI: 10.24432/C5R022

2. Feature Engineering

For each material, the elemental composition is represented as a fraction vector (summing to 1.0). These fractions are combined with atomic properties to produce 81 features:

featurei = Σj (fractionj × propertyj,i)

Properties include: atomic mass, atomic radius, electronegativity, ionization energy, electron affinity, density, thermal conductivity, specific heat, valence electrons, and number of unfilled orbitals.

3. Dimensionality Reduction

PCA (Principal Component Analysis) projects the 81-dimensional feature space to 2 components, retaining 73.5% of total variance (PC1: 56.2%, PC2: 17.4%). PCA captures linear correlations between compositional features.

UMAP (Uniform Manifold Approximation and Projection) provides a nonlinear embedding of the same 81D space into 2D, with parameters: n_neighbors=15, min_dist=0.1, cosine metric. UMAP better preserves local neighborhood structure, revealing composition families that PCA may compress.

Both projections are available as axis options. The third axis (Y) defaults to Tc, creating a 3D landscape where altitude corresponds to critical temperature.

4. Clustering

DBSCAN (Density-Based Spatial Clustering of Applications with Noise) identifies composition families without requiring a predefined cluster count. Parameters: eps=0.5, min_samples=10.

Result: 3 major clusters + outliers (cluster -1). Cluster assignments reflect compositional similarity in the full 81D space, not visual proximity in 2D projections.

5. Opportunity Scoring

Sparse regions in composition space are scored as potential discovery opportunities using four weighted factors:

Sparseness (40%)Inverse local density — fewer nearby materials indicates underexplored territory
Tc Proximity (30%)Mean critical temperature of 10 nearest neighbors — high values suggest the region may harbor high-Tc materials
Novelty (20%)Cluster diversity in local neighborhood — regions at cluster boundaries are compositionally unusual
Feasibility (10%)Penalty for extreme atomic radius mismatch — large mismatches hinder synthesis

Each factor is normalized to [0, 1] and combined as a weighted sum, scaled to 0–10. Scores are heuristic indicators, not ML-predicted synthesis success probabilities.

6. External Validation

Materials Project cross-reference: Individual materials can be enriched with data from the Materials Project database via API. When available, this provides:

  • Thermodynamic stability: Energy above convex hull (Ehull). Materials with Ehull = 0 are thermodynamically stable; Ehull < 0.1 eV/atom are metastable but potentially synthesizable; Ehull > 0.1 eV/atom are unlikely to form.
  • Synthesis verification: Materials with known crystal structures in Materials Project (nsites > 0) have been experimentally characterized or computationally verified.
  • Electronic structure: Band gap and formation energy data from DFT calculations.

Prediction error analysis: The Hamidieh dataset provides only experimental Tc measurements — no model predictions are included. A prediction-vs-experimental scatter plot is therefore not applicable to this dataset. Users seeking predictive models should apply regression techniques (e.g., Random Forest, XGBoost, neural networks) to the 81 compositional features with Tc as the target variable.

7. Limitations

  • Composition-only: No crystal structure data. CIF exports contain composition metadata only; use Materials Project for full structures.
  • Heuristic scoring: Opportunity scores reflect compositional analysis, not thermodynamic or structural predictions.
  • UMAP stochasticity: UMAP projections depend on random seed (fixed for reproducibility). Different seeds produce qualitatively similar but not identical layouts.
  • Dataset vintage: Hamidieh 2018 — materials discovered after 2018 are not represented.
  • No prediction model: Tc values shown are experimental measurements, not predictions. This tool does not predict Tc for new compositions.
